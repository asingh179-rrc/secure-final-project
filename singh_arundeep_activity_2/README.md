## Additional Information

The following Module 2 components were implemented in this activity:

1-Subclass Implementation – Created "Triangle" and "Rectangle" classes as subclasses of the "Shape" superclass, following proper inheritance rules and validation requirements.  
2-Package Initialization – Configured the "__init__.py" file inside the "shape" package to allow importing all classes with a single statement ("from shape import *").  
3-Client Program – Developed "activity_02_main.py" to instantiate shape objects, handle exceptions, and demonstrate polymorphism by calculating and displaying area and perimeter for different shapes.
